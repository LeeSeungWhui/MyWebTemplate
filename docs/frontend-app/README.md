# Frontend App Planning

앱(Expo / React Native) 기획안 진입점이다.
실제 스펙 원문은 아래 문서를 기준으로 유지한다.

## Core Docs
- 모듈 스펙: `docs/frontend-app/modules/app.md`
- RN 코딩 규칙: `docs/frontend-app/codding-rules-rn.md`
- 공통 규칙: `docs/common-rules.md`

## Unit Specs
- 경로: `docs/frontend-app/units/CU-APP-*.md`
- 범위: 로그인/가드/API 클라이언트/대시보드/오류 UX/재검증

## 운영 원칙
- 앱은 인증 기반 운영 화면 중심으로 구성한다.
- 공개 샘플 퍼널은 Web(`docs/frontend-web`)에서 담당한다.
