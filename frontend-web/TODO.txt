## 2. 샘플 페이지
### 기본 페이지 (필수) 🔥
1. 로그인/회원가입
   - 소셜 로그인
   - 이메일 인증
   - 비밀번호 찾기

2. 대시보드
   - 통계 차트
   - 알림/할일 목록
   - 최근 활동 내역

3. 게시판
   - 목록/상세/작성/수정
   - 파일 첨부
   - 댓글 기능

### 추가 페이지 (선택) ⭐
1. 관리자 페이지
   - 사용자 관리
   - 권한 관리
   - 시스템 설정

2. 마이페이지
   - 프로필 수정
   - 비밀번호 변경
   - 활동 내역

3. 설정 페이지
   - 테마 설정
   - 알림 설정
   - 개인정보 설정

4. 에러 페이지
   - 404 Not Found
   - 403 Forbidden
   - 500 Server Error
   - 점검 중 페이지
